## Featured Research Project Balancing Predictive Performance and Interpretability in Clinical Risk Models

This research evaluates whether interpretable statistical models can achieve comparable performance to modern machine learning methods in clinical risk prediction.

Methods:
Compared logistic regression, LASSO logistic regression, random forest, and XGBoost using cross-validation, ROC-AUC, calibration curves, Brier scores, SHAP-based interpretability, and statistical comparison using DeLong’s test.

Key Findings:

Logistic regression achieved AUC = 0.837, comparable to machine learning models
No statistically significant differences in model performance (p > 0.05)
Calibration performance was similar across all models
Interpretable models provide practical advantages for clinical decision-making
